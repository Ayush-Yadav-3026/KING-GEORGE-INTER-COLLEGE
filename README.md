Overview:
KGIC School Website is a responsive, PHP-powered website for King George Inter College. It showcases academic programs, events, campus life, student achievements, and provides information for prospective students and parents.

Technologies Used:

PHP
HTML5 & CSS3
JavaScript (for interactivity)


Key Features:

Responsive layout for desktop and mobile
Dynamic pages: About Us, Academic, Gallery, Contact Us, Blog/News
Event highlights and calendar
Admission form and downloadable prospectus
Student achievements, sports & cultural activities showcase
Contact and inquiry forms
