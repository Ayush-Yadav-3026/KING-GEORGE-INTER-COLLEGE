KGIC School Website is a responsive PHP and CSS based platform for King George Inter College. It provides information about academic programs, events, student achievements, and campus life. The site also includes contact and admission forms, making it easy for prospective students and parents to connect.

The front-end is built with HTML5 and CSS3 for a clean and modern design, while PHP and MySQL handle dynamic content and form submissions. JavaScript adds interactivity for smooth navigation and gallery features.

Check out the live site here: https://kgicschool.org.in/
